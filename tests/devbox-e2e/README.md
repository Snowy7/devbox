# Devbox E2E

End-to-end product tests and smoke harness fixtures belong here.

Use this area for flows such as login, share, clone, pause/resume, hosted storage, and multi-machine
simulation.

Current top-level smoke commands:

- `scripts/mvp-two-device-smoke` or `scripts/mvp-two-device-smoke.ps1` for the product MVP path.
- `scripts/alpha-workspace-adapters-smoke.ps1` for sparse folder intents, agent workspaces,
  materialized fallback, and filesystem adapter alpha evidence.
