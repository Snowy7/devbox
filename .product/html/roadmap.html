Roadmap

Earn trust before replacing workflows.

The roadmap starts with local snapshots, a local change feed, and two-device personal sync, then expands into team policies, agent sandboxes, and Loom-powered source-control UX.

Overview Market KPIs Roadmap MD

Phases

0
Prototype foundation. Local scanner, object store, manifests, restore, pending change feed, Git detector, destructive tests.
1
Personal alpha. Account pairing, watcher, encrypted sync, second-device materialization, CLI, secret block policy.
2
Trust beta. Timeline UI, policy editor, rehydration hints, restore quality, OS watcher hardening.
3
Paid Pro. Billing, storage limits, retention tiers, VS Code extension, docs.
4
Team preview. Managed policies, shared workspace links, device approval, audit, admin recovery.
5
Agent sandboxes. Isolated work, shared overlays, provenance, tests, merge or discard.
6
Loom layer. Checkpoints, sandboxes, Git import/export, PR publishing.
