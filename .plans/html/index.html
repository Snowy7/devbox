Build Stack

Layer Choice Reason
Core daemon Rust + Tokio Long-running, filesystem-heavy, concurrent, safety-critical local software.
Desktop app Electron + React + TypeScript Fast UI iteration, mature tray/update ecosystem, clean redacted control-panel boundary.
Local state SQLite + content-addressed files Transactional metadata with blobs stored as files, not database BLOBs.
Cloud Rust API + Postgres + R2 Postgres for metadata, R2 for encrypted immutable blob packs.
