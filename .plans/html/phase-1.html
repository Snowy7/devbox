Deliverables

Sync Backend

  • Rust API with device pairing and auth.
  • Postgres metadata for accounts, devices, projects, manifests, and cursors.
  • R2 blob backend behind a provider interface.
  • Encrypted immutable blob pack upload and download.

Client Sync

  • Upload queue, retry, resume, and backoff.
  • Download and materialization queue.
  • Device cursor reconciliation.
  • Conflict-as-divergent-snapshot handling.

CLI and Minimal UI

  • `devbox init`, `devices`, `sync`, `pause`, `restore`.
  • Minimal Electron shell for onboarding and status.
  • Tray state: idle, syncing, paused, warning.
  • Support bundle without source content by default.

Safety

  • Secrets blocked unless policy says otherwise.
  • Git verification after materialization.
  • Atomic writes with rollback.
  • Manual pause and project exclusion.
